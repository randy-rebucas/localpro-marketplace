# 1. Escrow Service Fees

Since LocalPro holds client funds before releasing to providers, you can charge **Escrow Handling Fees**.


### Escrow Protection Fee

**Purpose:** Covers payment security, dispute protection, and fund holding.

Typical models:

- **1% – 3% of transaction**

- OR **flat ₱10 – ₱50 per transaction**

Example:

Client books cleaning service = ₱2,000

    Service Price: ₱2,000
    Escrow Fee (2%): ₱40
    Total Paid by Client: ₱2,040

Benefits:

- fraud protection

- payment guarantee

- dispute mediation

Platforms using this model:

- Upwork

- Fiverr

- Airbnb

# 2. Payment Processing Fee

Payment gateways charge processing fees. You can **pass this partially to users**.

Example gateways in Philippines:

- GCash

- PayMaya

- Stripe

- PayMongo

Typical fee:

    2.5% – 3.5% + ₱15

LocalPro can structure:

Example:

    Processing Fee = 2%

# 3. Withdrawal Fee (Provider Side)

Charge providers when they **withdraw earnings**.

Example:

| Withdrawal Method  | Fee       |
| ------------------ | --------- |
| Bank transfer      | ₱15 – ₱25 |
| GCash payout       | ₱10 – ₱15 |
| Instant withdrawal | 1%        |

Example flow:

    Provider Wallet: ₱5,000
    Withdrawal fee: ₱20
    Provider receives: ₱4,980

***


# 4. Urgent Booking Fee

For **same-day or rush services**.

Example:

| Service urgency  | Fee  |
| ---------------- | ---- |
| Standard booking | free |
| Same day         | ₱50  |
| 2-hour rush      | ₱100 |

Example:

Client needs plumber immediately.

    Service price: ₱1,500
    Urgent fee: ₱100

# 5. Platform Service Fee (Client Side)

Some marketplaces charge a **client-side service fee**.

Example:

    5% platform service fee

Example:

    Service price: ₱2,000
    Platform fee: ₱100
    Total paid: ₱2,100

Then LocalPro still takes **provider commission**.

This is a **dual-sided marketplace model**.
